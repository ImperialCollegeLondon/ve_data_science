# Using the HPC

To get started with the high performance computing (HPC) cluster at Imperial, first [register](https://www.imperial.ac.uk/admin-services/ict/self-service/research-support/rcs/get-access/) for access to it.

## Resources and documentation

Once you have gained access, it is recommended to browse through the [tutorials](https://www.imperial.ac.uk/admin-services/ict/self-service/research-support/rcs/get-support/training/), especially the [introductory slides](https://imperialcollegelondon.app.box.com/s/kwjxbd5bc87w296wo0m7fdwo9jct5vvs).

Once you have gotten an overview, browse the Research Computing Service (RCS) [documentation](https://icl-rcs-user-guide.readthedocs.io/en/latest/hpc/getting-started/). Below is a summarised sequence of set up:

## Further setup

- Set up [Globus](docs/using_globus.md)
